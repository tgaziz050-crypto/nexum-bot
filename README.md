# NEXUM v1.0 — AI Telegram Bot

Модель: nexum v1.0. Имя: NEXUM.

## Быстрый старт

1. `pip install -r requirements.txt`
2. Установи ffmpeg и yt-dlp (для видео/аудио)
3. Скопируй `config.example.env` в `.env` и заполни `BOT_TOKEN`
4. `python bot.py`

## Переменные (.env)

- `BOT_TOKEN` — обязательно
- `G1`, `GR1`, `DS1` — AI (нужен хотя бы один)
- `AUDD_API_KEY` — распознавание музыки
- `NOTION_API_KEY`, `NOTION_DATABASE_ID` — интеграция с Notion
- `ADMIN_IDS` — ID админов через запятую (согласование опасных действий)
- `REPLICATE_API_TOKEN` — опционально для видео/музыки

**Важно:** `.env` не попадает в git — API-ключи в безопасности.

## Возможности

Управление через естественный язык — без кнопок. Просто пиши, что нужно:

- Генерация изображений, видео, музыки
- Создание страниц в Notion
- Публикация в несколько каналов
- Поиск в интернете, погода, TTS
- Скачивание с YouTube/TikTok

Опасные действия (удаление сообщений, бан и т.п.) уходят на согласование админу в личку.
