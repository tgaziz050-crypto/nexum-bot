# 🤖 NEXUM — Telegram AI Bot

## 📁 Структура файлов
```
nexum/
├── bot.py              ← Главный код бота
├── requirements.txt    ← Библиотеки Python
├── .env.example        ← Пример настроек (скопируй в .env)
├── .env                ← Твои секретные ключи (создай сам)
├── memory.json         ← Память бота (создаётся автоматически)
├── start.bat           ← Запуск на Windows
└── start.sh            ← Запуск на Mac/Linux
```

---

## 🚀 ПОШАГОВАЯ ИНСТРУКЦИЯ

### ШАГ 1 — Получи токен бота
1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя: `NEXUM`
4. Придумай username: `nexum_ai_bot` (должен заканчиваться на `bot`)
5. Скопируй токен вида: `7412345678:AAH...`

### ШАГ 2 — Получи ключ Claude AI (бесплатно!)
1. Зайди на **console.anthropic.com**
2. Зарегистрируйся (даётся $5 кредитов бесплатно)
3. Перейди в **API Keys** → **Create Key**
4. Скопируй ключ вида: `sk-ant-api03-...`

### ШАГ 3 — Настрой .env файл
Создай файл `.env` в папке nexum и вставь:
```
BOT_TOKEN=7412345678:AAH_твой_токен_здесь
ANTHROPIC_KEY=sk-ant-api03-твой_ключ_здесь
```

### ШАГ 4 — Установи библиотеки
Открой командную строку (cmd) в папке nexum и напиши:
```
pip install -r requirements.txt
```

### ШАГ 5 — Запусти бота
**На Windows:**
```
start.bat
```
или
```
python bot.py
```

**На Mac/Linux:**
```
bash start.sh
```
или
```
python3 bot.py
```

### ШАГ 6 — Проверь!
Открой Telegram → найди своего бота → напиши `/start`
Должен ответить! 🎉

## 🌐 КАК ЗАПУСТИТЬ 24/7 БЕСПЛАТНО

### Вариант 1 — Railway (рекомендую)
1. Зарегистрируйся на **railway.app** через GitHub
2. Нажми **New Project** → **Deploy from GitHub**
3. Загрузи файлы на GitHub (или используй Railway CLI)
4. В настройках добавь переменные окружения:
   - `BOT_TOKEN` = твой токен
   - `ANTHROPIC_KEY` = твой ключ
5. Deploy! Бот работает 24/7

### Вариант 2 — Запуск на своём ПК
Просто оставь `python bot.py` работать в терминале.
Бот будет онлайн пока компьютер включён.

## ❓ Частые ошибки

**"ModuleNotFoundError"** → запусти `pip install -r requirements.txt`

**"Unauthorized"** → проверь BOT_TOKEN в .env файле

**"Invalid API Key"** → проверь ANTHROPIC_KEY в .env файле

**Бот не отвечает** → убедись что `python bot.py` запущен и работает
