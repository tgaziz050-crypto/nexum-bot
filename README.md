# NEXUM v5.0 — Запуск

## Быстрый старт

1. Установи зависимости:
   pip install -r requirements.txt

2. Установи ffmpeg (для видео/аудио):
   - Windows: https://ffmpeg.org/download.html (добавь в PATH)
   - Linux: apt install ffmpeg
   - Mac: brew install ffmpeg

3. Запусти:
   python bot.py

## Деплой на Railway/Render (бесплатно)

1. Создай аккаунт на railway.app
2. Новый проект → Deploy from GitHub
3. Загрузи файлы: bot.py + requirements.txt
4. В Settings → Variables добавь:
   BOT_TOKEN = твой_токен_бота
