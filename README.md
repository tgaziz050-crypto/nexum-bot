# NEXUM v2.0 — Сверхинтеллектуальный AI Telegram бот

## 🚀 Что нового в v2.0

### Улучшенный интеллект
- **Gemini 2.0 Flash Experimental** — умнее стандартной версии
- **Умная система памяти** — категоризация фактов, важность, контекст
- **Анализ эмоций** — бот понимает настроение пользователя
- **Адаптивное общение** — стиль меняется в зависимости от истории

### Новые возможности
- 📥 **Скачивание с YouTube** через yt-dlp (аудио/видео)
- 🔍 **Продвинутый поиск** в интернете
- 🔗 **Чтение любых ссылок** — извлечение текста со страниц
- 🎨 **Генерация изображений** через Pollinations Flux
- ⏰ **Напоминания** с планировщиком
- 💱 **Курсы валют** в реальном времени
- 🧮 **Калькулятор**
- 🌤 **Погода** с детальной информацией

### Работа с медиа
- 📷 **Анализ фото** через Gemini Vision
- 📹 **Анализ видео** — кадр + транскрипция речи
- 🔵 **Кружочки** — полная поддержка
- 🎤 **Голосовые** — транскрипция через Whisper
- 📎 **Документы** — чтение текстовых файлов

## 📋 Установка

### Railway (рекомендуется)

1. Форкни репозиторий на GitHub
2. Зайди на [railway.app](https://railway.app)
3. New Project → Deploy from GitHub
4. Выбери репозиторий
5. Добавь переменные окружения:

```
BOT_TOKEN=твой_токен_бота
GEMINI_1=AIzaSy...
GEMINI_2=AIzaSy...
GEMINI_3=AIzaSy...
GEMINI_4=AIzaSy...
GEMINI_5=AIzaSy...
GEMINI_6=AIzaSy...
GROQ_1=gsk_...
GROQ_2=gsk_...
GROQ_3=gsk_...
GROQ_4=gsk_...
GROQ_5=gsk_...
GROQ_6=gsk_...
```

6. Deploy!

### Локально

```bash
# Установи зависимости
pip install -r requirements.txt

# Установи ffmpeg и yt-dlp
# Ubuntu/Debian:
sudo apt install ffmpeg
pip install yt-dlp

# Запусти
python bot.py
```

## 🎯 Специальные маркеры (AI использует автоматически)

| Маркер | Функция |
|--------|---------|
| `%%IMG%%описание%%` | Генерация изображения |
| `%%WEB%%запрос%%` | Поиск в интернете |
| `%%URL%%ссылка%%` | Чтение веб-страницы |
| `%%WTR%%город%%` | Погода |
| `%%YT_DL%%ссылка%%` | Скачать с YouTube |
| `%%RATE%%USD%%RUB%%` | Курс валют |
| `%%CALC%%выражение%%` | Калькулятор |
| `%%REMIND%%минуты%%текст%%` | Напоминание |
| `%%REMEMBER%%факт%%` | Запомнить факт |

## 🧠 Система памяти

### Категории фактов
- **identity** — имя, возраст, пол
- **location** — город, страна
- **work** — профессия, работа
- **education** — учёба
- **interests** — хобби, увлечения
- **relationships** — семья, отношения
- **goals** — цели, мечты
- **problems** — проблемы, трудности
- **skills** — навыки, умения

### Автоматическое извлечение
Бот автоматически извлекает факты из разговора и сохраняет их с приоритетом важности.

## 🔄 Архитектура AI

```
Запрос пользователя
        ↓
┌─────────────────────┐
│ Gemini 2.0 Flash Exp │ ← Основной (самый умный бесплатный)
└─────────┬───────────┘
          ↓ если не отвечает
┌─────────────────────┐
│ Gemini 2.0 Flash    │ ← Fallback
└─────────┬───────────┘
          ↓ если не отвечает
┌─────────────────────┐
│ Groq Llama 3.3 70B  │ ← Последний резерв
└─────────────────────┘
```

### Ротация ключей
6 Gemini ключей × 1500 запросов = **9000 запросов/день**
6 Groq ключей — для STT и fallback

## 📊 База данных

SQLite с таблицами:
- `users` — профили пользователей
- `conversations` — история с метаданными
- `memories` — долгосрочная память
- `preferences` — настройки
- `context` — текущий контекст разговора
